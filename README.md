# Utility Functions MVC App

This ASP.NET Core MVC project demonstrates three small utilities:
- **Vowel Counter**
- **Leap Year Checker**
- **Palindrome Checker** (works with both numbers and strings)

---

## 🧰 Prerequisites

Before running, make sure you have:

- **.NET SDK 8.0 or later**  
  You can verify with:
  ```bash
  dotnet --version
